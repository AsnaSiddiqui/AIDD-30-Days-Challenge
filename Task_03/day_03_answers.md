## Research Questions & Answers

### 1. What new improvements were introduced in Gemini 3.0?

Gemini 3.0 Google ka ab tak ka sab se advanced model hai.Iski reasoning pehlay se bohot strong ho gayi hai, aur yeh text, images, videos sab ek sath samajh sakta hai.
Isme agent wali abilities bhi add ho gayi hain, jisse yeh multi-step kaam khud se kar leta hai.
Coding, speed aur overall performance bhi improve ho gayi hai, is liye real projects me bohot powerful feel hota hai.


### 2. How does Gemini 3.0 improve coding & automation workflows?

Gemini 3.0 coding aur automation dono ko bohot smooth kar deta hai, kyunki isme agent-style kaam karne ki ability aa gayi hai.
Yeh multi-step tasks khud handle karta hai aur natural language se proper code, UI aur workflows generate kar deta hai.
Gemini 3.0 ko Images, videos aur sketches ko samajh kar code banana bohot easy ho gaya hai, aur VS Code, GitHub, aur Gemini CLI jaisay tools ke sath iska integration bhi strong ho chuka hai.


### 3. How does Gemini 3.0 improve multimodal understanding?

Gemini 3.0 ki multimodal ability pehlay se bohot strong ho gayi hai, kyunki yeh text, images, audio aur video ko ek sath naturally samajh leta hai.
Videos aur long audio ko analyze karna, important moments pick karna, aur unka meaning samajhna ab kaafi easy ho gaya hai.
Yeh visuals ko reasoning ke sath combine karke smarter answers deta hai, aur screenshots ya designs ko dekh kar code bhi generate kar sakta hai.


### 4. Name any two developer tools introduced with Gemini 3.0.?

Gemini 3.0 ne developers ke liye kuch strong tools introduce kiye hain, jin me Google Antigravity aur Jules bohot important hain.

Antigravity: AI agents ko full development tasks handle karne deta hai, jaise planning, coding aur fixes.

Jules: Ek coding agent hai jo GitHub repository ke sath kaam karta hai aur background me issues solve karke pull requests bana deta hai.
